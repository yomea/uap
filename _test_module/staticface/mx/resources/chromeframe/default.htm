谷歌浏览器内嵌框架是一款免费的 Internet Explorer 插件。某些高级网络应用程序就是采用谷歌浏览器内嵌框架为您提供多元化的功能和优良的性能。

下载 Google Chrome 浏览器内嵌框架

安装快捷，加速访问。

适用于 Windows 7/Vista/XP SP2 或更高版本系统上的 Internet Explorer 6、7、8、9。

  1. 单击此页面右上角的“下载 Google Chrome 浏览器内嵌框架” 按钮，以开始下载。
  2. 可以执行下列操作之一：
    • 若要立即开始安装，请单击“运行”。
    • 若要将下载的文件保存到计算机上，以后再执行安装，请单击“保存”。
    • 若要取消安装，请单击“取消”。
    • 安装浏览器内嵌框架之后，您需要重新启动浏览器。
重要事项： 如果完成上述操作后，仍然不能正常访问系统，则必须在 IE 的工具栏菜单中，选择 “Internet 选项 -> 高级-> 浏览”， 勾选“启用第三方浏览器扩展” 。然后再关闭窗口或浏览器，刷新当前页面。
